Postmark overwrites wp_mail to send emails via Postmark. You will need a Postmark account to make use of this plugin.

== Description ==

Postmark plugin for WordPress overrides WordPress' wp_mail function to make use of Postmark's virtual servers to send emails. Don't want the hassle of setting up your own SMTP server then Postmark is a great solution that is easy to configure and set up within just a few minutes.

== Installation ==

1. Upload postmark directory to your /wp-content/plugins directory
2. Activate plugin in WordPress admin
3. In WordPress admin, go to Settings then Postmark. You will then want to insert your Postmark details.
4. Test the settings using the "Send Test Email" button.
5. If happy with the integration then "Enable" the override of wp_mail.
